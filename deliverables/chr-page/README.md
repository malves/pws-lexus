# CHR Page Package

Contenu de ce dossier :

- `chr.html` : page Toyota C-HR+ a copier sur la page `chr.html` du repo cible
- `index.html` : copie identique pour previsualiser la page seule si besoin
- `assets/` : seulement les assets utilises par cette page

Pour l'integrer dans le repo GitHub cible :

1. Copier `chr.html` a la racine du repo cible
2. Copier les fichiers de `assets/` dans le dossier `assets/` du repo cible
3. Si tu veux une previsualisation standalone, tu peux aussi ouvrir `index.html`

Note :

- La video du hero reste servie depuis Toyota via une URL distante.
- Les images de contenu sont locales dans `assets/`.
